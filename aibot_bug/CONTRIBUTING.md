# Contributing to AIbot-bug

First off, thank you for considering contributing to AIbot-bug! It's people like you who make it such a powerful tool for the community.

### Reporting Bugs
*   Check the [Issues](https://github.com/yourusername/AIbot-bug/issues) to see if it's already been reported.
*   If not, open a new issue with a clear description and steps to reproduce.

### Suggesting Enhancements
*   We love new "Beast Mode" features! Open an issue with the [Enhancement] tag.

### Pull Requests
1.  Fork the repo.
2.  Create a new branch (`git checkout -b feature/AmazingFeature`).
3.  Commit your changes (`git commit -m 'Add some AmazingFeature'`).
4.  Push to the branch (`git push origin feature/AmazingFeature`).
5.  Open a Pull Request.

## üìú Coding Guidelines
*   **Modularity:** Keep new tools integrated via `run_tool` and `AIManager`.
*   **Error Handling:** Use the `ReliabilityManager` for network-heavy tasks.
*   **Documentation:** Update the `TOOL_USAGE` dictionary in `Newpro.py` for any new tools.
